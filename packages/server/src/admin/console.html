YAML EINFÜGEN

Generiere eine Quest mit der QUEST-ENGINE (siehe QUEST_ENGINE.md) und füge das YAML hier ein.

YAML FORMAT (Referenz)
version: '1.0'
quest:
  title: 'Die Verlorene Ladung'
  description: 'Sammle Ressourcen und liefere sie ab.'
  scope: universal          # universal | individual | sector
  type: fetch               # fetch | delivery | scan | bounty | custom
  npc_name: 'Commander Vex'
  npc_faction: pirates      # traders | scientists | pirates | ancients | independent
  expires_days: 7
  max_acceptances: 50       # optional
  sector:                   # nur für scope: sector
    x: 12
    y: -5
  target_players:           # nur für scope: individual (Usernames)
    - PlayerOne
    - PlayerTwo
  objectives:
    - type: fetch
      description: '50 Erz sammeln'
      resource: ore
      amount: 50
    - type: delivery
      description: 'Zur Basisstation liefern'
      target_x: 5
      target_y: 3
  rewards:
    credits: 5000
    xp: 200
    reputation:
      faction: traders
      amount: 15
  flavor:
    intro_text: 'Pilot, ich brauche deine Hilfe...'
    completion_text: 'Ausgezeichnet!'
