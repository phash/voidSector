## Kapitel 1: Was ist das hier eigentlich?

Herzlichen Glückwunsch! Du hast gerade ein Schiff. Ein kleines, unpersönliches, völlig ahnungsloses Schiff. Es kennt dich nicht. Es mag dich nicht. Es ist hauptsächlich damit beschäftigt, seine eigenen Systeme nicht zu vergessen.

Das wird sich ändern.

Das **Adaptive Craft Evolution Protocol** — kurz **ACEP**, lang "das Ding, das dein Schiff seltsam macht" — ist das System, das dafür sorgt, dass dein Schiff mit der Zeit zu *deinem* Schiff wird. Nicht weil du Punkte verteilt hast. Nicht weil du ein Menü geöffnet hast. Sondern weil du *Dinge getan* hast, und dein Schiff dabei zugeschaut und gelernt hat.

```
┌────────────────────────────────────────────────────────┐
│                                                        │
│   SCHIFF (NEU):                                        │
│                                                        │

## Kapitel 2: Das XP-Budget — Oder: Warum du nicht alles haben kannst

ACEP hat vier Pfade. Dein Schiff hat **100 XP-Punkte** total, maximal **50 pro Pfad**.

Das klingt viel. Es ist nicht viel.

```
         DEIN GESAMTES ACEP-BUDGET
         ╔══════════════════════════╗
         ║                          ║
         ║  AUSBAU  │  INTEL        ║
         ║  [░░░░░] │ [░░░░░]        ║
         ║  max 50  │ max 50        ║
         ║──────────┼───────────────║
         ║  KAMPF   │  EXPLORER     ║
         ║  [░░░░░] │ [░░░░░]        ║
         ║  max 50  │ max 50        ║
         ╚══════════════════════════╝
              TOTAL: max 100
```

**Was bedeutet das?**

Wenn du 50 XP in KAMPF steckst, hast du noch 50 übrig. Du kannst nicht gleichzeitig auch 50 in INTEL haben. Du musst wählen. Das Universum zwingt dich zur Persönlichkeit.

> **WARNUNG:** Spieler, die versucht haben, alle 4 Pfade gleichzeitig zu maximieren, wurden zuletzt beim Rechnen von `4 × 50 = 100` gesehen. Das stimmt zwar. Aber `50 + 50 + 0 + 0 = 100` auch. Überlege genau, wen du bist.

**Typischer AUSBAU-Pilot:** Versteht Piraten nicht. Fragt sich, warum die nicht einfach Fracht transportieren. Hat seine Base sehr schön eingerichtet. Cargo ist sein Lebenssinn.

## Kapitel 5: Das Schiff entwickelt eine Persönlichkeit — Und das ist seltsam

Nach einer Weile beginnt dein Schiff, Dinge zu sagen. Das ist normal. Bitte erschrick nicht.

Das System heißt **Persönlichkeitsmeldungen** und dein Schiff kommentiert Situationen basierend auf seinen gesammelten Erfahrungen.

```
JUNGES SCHIFF (0 XP):
┌────────────────────────────────────┐
│ [SYSTEM]: Scan abgeschlossen.      │
│ [SYSTEM]: Keine Anomalien.         │
│ [SYSTEM]: ...                      │  ← Es hat noch nicht viel zu sagen.
└────────────────────────────────────┘

VETERAN-SCHIFF (kampf ≥ 20):
┌────────────────────────────────────┐
│ [SYSTEM]: Piratensignal.           │
│ [SYSTEM]: Bekannte Taktik.         │
│ [SYSTEM]: Routine.                 │  ← Es hat Meinungen.
└────────────────────────────────────┘

ANCIENT-TOUCHED-SCHIFF (explorer ≥ 15):
┌────────────────────────────────────┐
│ [SCHIFF]: Diese Leere...           │
│ [SCHIFF]: Wir waren schon hier.    │
│ [SCHIFF]: In einer anderen Form.   │  ← Jetzt macht es ein bisschen Angst.
└────────────────────────────────────┘

RECKLESS-SCHIFF (kampf ≥ 15, ausbau ≤ 5):
┌────────────────────────────────────┐
│ [SYSTEM]: Scan durch.              │
│ [SYSTEM]: Kein Kontakt.            │
│ [SYSTEM]: Schade.                  │  ← Das Schiff ist enttäuscht, dass niemand kämpft.
└────────────────────────────────────┘
```

> **Hinweis:** Wenn dein Schiff beginnt, Dinge zu sagen wie *"Sie sind nicht weg. Nur woanders."*, ist das ein Ancient-Touched-Trait. Das Schiff hat zu viele Ruinen gescannt. Das ist kein Bug. Es geht ihm gut. Wahrscheinlich.

## Kapitel 6: Das Permadeath-Problem

Dein Schiff kann sterben. Das ist kein Fehler. Das ist ein Feature.

```
HP:  ████████████████████  100%   ← Gut

HP:  █████               25%    ← Weniger gut

HP:  █                    5%    ← EJECT-POD VERFÜGBAR!

HP:                       0%    ← Dein Schiff ist jetzt ein Wrack.
                                   Es treibt irgendwo im Void.
                                   Andere Piloten können es finden.
                                   Das ist sein Erbe.
```

### Was passiert nach dem Tod?

Dein Nachfolger-Schiff erbt:
- **30 % deiner ACEP-XP** (verteilt auf alle Pfade)
- **1 Trait** des Vorgängers
- Die Erinnerung daran, dass das Universum gefährlich ist

```
VORGÄNGER:        NACHFOLGER:
AUSBAU: 50     → AUSBAU: 15
INTEL:  30     → INTEL:   9
KAMPF:   0     → KAMPF:   0
EXPLORER: 20   → EXPLORER: 6

Vererbter Trait: cautious
(Das neue Schiff hat gelernt, dass Vorsicht überlebt.)
```

### Das Wrack

Dein totes Schiff bleibt im Universum. Irgendwo bei Koordinaten X/Y existiert jetzt ein Pixel deiner Geschichte.

```
