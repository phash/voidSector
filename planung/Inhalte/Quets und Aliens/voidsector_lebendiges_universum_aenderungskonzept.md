# VoidSector -- Änderungskonzept: Lebendiges Universum

## Ziel der Änderung

Das Universum soll sich dynamisch entwickeln statt statisch zu bleiben.

Neue Ziele: - Alienrassen expandieren aktiv - Territorien entstehen
organisch - Spieler beeinflussen die Expansion der Menschheit -
NPC-Fraktionen agieren glaubwürdig - Karten zeigen sichtbare politische
Veränderungen

Das Ergebnis ist ein lebendiges Strategie-Ökosystem im Hintergrund.

------------------------------------------------------------------------

# 1. Fraktionen / Gruppen

Alle intelligenten Spezies werden als Gruppen (Factions) behandelt.

Beispiele: - Menschen - Archivare - K'thari - Mycelianer -
Handels-Konsortium

Spieler starten immer als Mitglied der Menschheitsgruppe.

Menschen sind die einzige Gruppe mit vielen Spielern. Andere Fraktionen
bestehen hauptsächlich aus NPC-Flotten und Stationen.

------------------------------------------------------------------------

# 2. Universumsstruktur

Das Universum besteht aus Quadranten.

Ein Quadrant enthält: - Asteroiden - Planeten - Ressourcenfelder -
Stationen - Schiffe

## Startzustand

Zu Beginn besitzt die Menschheit 5 Quadranten.

Außerhalb dieser Quadranten gibt es: - keine menschlichen Stationen -
keine menschliche Infrastruktur

Alienfraktionen besitzen bereits verstreute Gebiete.

------------------------------------------------------------------------

# 3. Territoriumssystem

Die Kontrolle eines Quadranten basiert auf Stationen.

Wenn eine Gruppe Stationen baut: → wird der Quadrant auf der Karte
farblich markiert.

Wenn mehrere Gruppen Stationen besitzen: → wird der Quadrant anteilig
eingefärbt.

Beispiel: - 70% K'thari - 30% Menschen

Berechnung kann erfolgen über: - Anzahl der Stationen - Stationslevel -
Infrastrukturwert

------------------------------------------------------------------------

# 4. Expansion der Fraktionen

Expansion basiert auf einem wirtschaftlichen Kreislauf.

1.  Ressourcen scannen
2.  Mining-Schiffe bauen Ressourcen ab
3.  Transport zu Stationen
4.  Station produziert neue Einheiten
5.  Frachter gründen neue Kolonien

Beispielablauf:

Station produziert Frachter\
↓\
Frachter wird mit Ressourcen beladen\
↓\
Frachter fliegt zu neuem Quadranten\
↓\
Frachter baut neue Station\
↓\
Neue Expansion beginnt

------------------------------------------------------------------------

# 5. Unterschied Menschen vs Alienfraktionen

## Alienfraktionen

-   stark automatisiert
-   viele NPC-Schiffe
-   stabile Wirtschaft

## Menschen

-   nur wenige automatische NPCs
-   Expansion hängt stark von Spielern ab

Spieler müssen: - Ressourcen abbauen - Stationen bauen - Frachter
schützen

Dies fördert Kooperation zwischen Spielern.

------------------------------------------------------------------------

# 6. Jumpgate-System

Jumpgates verbinden zwei Quadranten.

Funktion: - schnellere Reisen - strategische Handelsrouten -
militärische Mobilität

NPCs können Jumpgates: - bauen - nutzen

------------------------------------------------------------------------

# 7. Sichtbarkeit von Schiffen

Freundliche NPC-Schiffe sind immer im gesamten Quadranten sichtbar.

Feindliche Schiffe bleiben sensorabhängig sichtbar.

Das sorgt für: - sichtbare Handelsrouten - mehr Leben auf der Karte -
erkennbare Flottenbewegungen

------------------------------------------------------------------------

# 8. Geteilte Sensorinformationen

Alle Mitglieder einer Gruppe teilen ihre Scannergebnisse.

Daten werden pro Tick synchronisiert.

Wenn ein Spieler scannt: → sehen alle Menschen diese Information.

Alienfraktionen teilen ihre Daten ebenfalls intern.

------------------------------------------------------------------------

# 9. Tick-basiertes Universum

Das Universum läuft in festen Zeitintervallen.

1 Tick = 5 Sekunden

Pro Tick passieren: - NPC Bewegungen - Ressourcenabbau - Produktion -
Sensorupdates - Kampfberechnungen

------------------------------------------------------------------------

# 10. Aktionspunkte-System (AP)

Jeder Spieler erhält pro Tick:

5 Aktionspunkte

Beispiele:

  Aktion       AP
  ------------ ------
  Bewegen      1
  Mining       1
  Scannen      1
  Reparieren   2
  Bauen        3--5

Wenn eine Aktion mehr AP benötigt als verfügbar: → wird sie über mehrere
Ticks ausgeführt.

------------------------------------------------------------------------

# 11. Mining-System

Mining kostet:

1 AP pro Abbauaktion

Der Ertrag basiert auf:

Spieler Mining Level + Mining Ausrüstung des Schiffs

Beispiele:

Frühes Spiel: 1 Ressource pro AP

Mittleres Spiel: 5 Ressourcen pro AP

Spätes Spiel: 50 Ressourcen pro AP

------------------------------------------------------------------------

# 12. NPC Bewegungssystem

NPC-Schiffe bewegen sich tickbasiert.

Beispiel:

Tick 1 → Bewegung starten\
Tick 2 → Position aktualisieren\
Tick 3 → Ziel erreicht

Dies ermöglicht synchrones Multiplayer-Gameplay.

------------------------------------------------------------------------

# 13. Sichtbare Wirtschaft

Spieler können beobachten:

-   Mining-Schiffe
-   Frachter
-   Stationsbau

Dies erzeugt das Gefühl einer lebendigen Wirtschaft.

Beispiel:

6 Frachter fliegen zur Station → Station produziert neue Schiffe → neue
Kolonie entsteht.

------------------------------------------------------------------------

# 14. Spieleraufgaben

Spieler unterstützen die Menschheit durch:

## Wirtschaft

-   Mining
-   Transport
-   Stationsbau

## Militär

-   Konvoischutz
-   Alienabwehr
-   Sektorverteidigung

## Exploration

-   neue Ressourcen entdecken
-   neue Quadranten scannen

------------------------------------------------------------------------

# 15. Langfristige Effekte

Dieses System erzeugt:

## Dynamische Galaxie

Gebiete können: - erobert werden - verloren gehen - neu besiedelt werden

## Spielerentscheidungen zählen

Wenn Spieler expandieren: → Menschheit wächst.

Wenn nicht: → Alienfraktionen dominieren.

## Emergent Gameplay

Neue Situationen entstehen:

-   Handelskriege
-   Alieninvasionen
-   wirtschaftliche Krisen

------------------------------------------------------------------------

# 16. Vorteile des Systems

-   Universum fühlt sich lebendig an
-   Spieler haben echten Einfluss
-   Expansion entsteht organisch
-   Fraktionen wirken glaubwürdig
-   ideal für Multiplayer
